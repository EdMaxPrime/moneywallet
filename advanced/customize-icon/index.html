Customize an Icon

Money Wallet uses material design avatars. They are circles next to things like categories, events, wallets, etc. By default, it uses the first letter of the first 2 words in the name and a black background.

There are 2 types of icons:

  1. Initials (first letter of the first 2 words, with a background color)
  2. Picture

You can customize it by clicking "edit" on anything with an icon (category, event, wallet, etc). A drawer will slide up from the bottom of the screen asking you to choose between the two types of icons.

If you stick with the Initials (background color) style icon, the letters can only be changed by changing the name or editing the JSON export.
